# LauncherableRomsz
Launcherable but not Video games! It's Retro Games!



Mostly EmulatorJS



To add games, fork this repo, in the Games.json add

 "title": "Title",
    "entry": "games/Gamefolder/HTML file"


Then, In the Games folder add a folder make sure the Game Folder name and the Name of the game folder in the Entry in your Games.json match, otherwise, it will not work.

In the Game folder you need:

a index.html file
